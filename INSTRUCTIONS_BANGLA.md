# Projukti X - লোকাল কম্পিউটারে রান করার নিয়মাবলী 🚀

এই ওয়েবসাইটটি যেকোনো কম্পিউটারে কোনো ধরনের সমস্যা ছাড়াই রান করানোর জন্য নিচের ধাপগুলো অনুসরণ করুন।

## ১. প্রয়োজনীয় সফটওয়্যার ইনস্টল করা (যদি না থাকে)
আপনার কম্পিউটারে যদি **Node.js** ইনস্টল করা না থাকে, তাহলে প্রথমে সেটি ইনস্টল করতে হবে।
- যেকোনো ব্রাউজার থেকে [Node.js এর অফিসিয়াল ওয়েবসাইটে](https://nodejs.org/) যান।
- সেখান থেকে **"LTS" (Long Term Support)** ভার্সনটি ডাউনলোড করে সাধারণ সফটওয়্যারের মতো ইনস্টল করে নিন।

## ২. প্রজেক্ট ফাইলগুলো প্রস্তুত করা
- আপনি যে ফোল্ডারটি (প্রজেক্ট ফাইল) পেয়েছেন, সেটি আপনার কম্পিউটারের যেকোনো একটি ড্রাইভে রাখুন। 
- ফোল্ডারটির ভেতরে প্রবেশ করুন (যেখানে `package.json`, `src`, `public` ইত্যাদি ফাইলগুলো আছে)।

## ৩. প্রজেক্ট চালু করার নিয়ম
১. ফোল্ডারটির ভেতরে থাকা অবস্থায় উপরের অ্যাড্রেস বারে (Address bar) ক্লিক করুন এবং লিখুন **`cmd`**। তারপর **Enter** চাপুন। এতে একটি কালো টার্মিনাল (Command Prompt) ওপেন হবে।
২. টার্মিনালে নিচের কমান্ডটি লিখে **Enter** চাপুন:
   ```bash
   npm install
   ```
   *(এটি প্রজেক্টের সব প্রয়োজনীয় ফাইলগুলো ইন্টারনেট থেকে ডাউনলোড করবে। এর জন্য আপনার কম্পিউটারে ইন্টারনেট কানেকশন থাকতে হবে। এটি মাত্র একবার করতে হবে।)*

৩. ডাউনলোড শেষ হলে, ওয়েবসাইটটি রান করার জন্য নিচের কমান্ডটি লিখে **Enter** চাপুন:
   ```bash
   npm run dev
   ```

৪. একটু পরেই টার্মিনালে একটি লিংক দেখতে পাবেন (সাধারণত: `http://localhost:5173/`)।
৫. আপনার কিবোর্ডের **Ctrl** বোতাম চেপে ধরে ওই লিংকে ক্লিক করুন অথবা লিংকটি কপি করে আপনার ব্রাউজারে (Chrome, Edge) পেস্ট করে **Enter** চাপুন।

ব্যাস! আপনার ওয়েবসাইটটি সম্পূর্ণ অফলাইনে আপনার লোকাল কম্পিউটারে রান হয়ে যাবে! 🎉

---

**অতিরিক্ত তথ্য:**
- ওয়েবসাইটটি বন্ধ করতে চাইলে টার্মিনালে গিয়ে **Ctrl + C** চাপুন এবং **Y** লিখে এন্টার দিন।
- পরবর্তীতে যখনই ওয়েবসাইটটি আবার চালু করতে চাইবেন, তখন শুধু ফোল্ডারে গিয়ে `cmd` ওপেন করে `npm run dev` কমান্ডটি দিলেই হবে (`npm install` আর দেয়ার প্রয়োজন নেই)।
