# Projukti X - Local Run Manual (প্রযুক্তি এক্স - লোকাল রান ম্যানুয়াল)

এই প্রজেক্টটি যেকোনো কম্পিউটারে লোকাল সার্ভারে রান করার জন্য নিচের ধাপগুলো অনুসরণ করুন।

## ১. প্রয়োজনীয় সফটওয়্যার ইনস্টলেশন (Requirements)

যেকোনো কম্পিউটারে এটি রান করতে হলে প্রথমে **Node.js** ইনস্টল থাকতে হবে। 

- প্রথমে [Node.js এর অফিসিয়াল ওয়েবসাইট (https://nodejs.org/)](https://nodejs.org/) এ যান।
- সেখান থেকে **LTS (Long Term Support)** ভার্সনটি ডাউনলোড করে সাধারণ সফটওয়্যারের মতো ইনস্টল করে নিন।

## ২. প্রজেক্ট রান করার নিয়ম (How to run)

প্রজেক্টটি রান করার জন্য আপনাকে টার্মিনাল বা কমান্ড প্রম্পট (Command Prompt / PowerShell / VS Code Terminal) ব্যবহার করতে হবে।

ধাপগুলো হলো:

১. প্রথমে প্রজেক্টের মেইন ফোল্ডার (যেখানে `package.json` ফাইলটি আছে) সেখানে যান।
২. সেই ফোল্ডারের অ্যাড্রেস বারে `cmd` লিখে `Enter` চাপুন (এতে ওই ফোল্ডারে কমান্ড প্রম্পট ওপেন হবে) অথবা VS Code দিয়ে ফোল্ডারটি ওপেন করে টার্মিনাল খুলুন।
৩. এবার নিচের কমান্ডটি লিখে `Enter` চাপুন (এটি প্রজেক্টের প্রয়োজনীয় সব ফাইল ইন্টারনেট থেকে ডাউনলোড করে সেটআপ করবে, এটি শুধু একবার করতে হয়):

```bash
npm install
```

৪. ইনস্টলেশন শেষ হলে ওয়েবসাইটটি চালু করার জন্য নিচের কমান্ডটি লিখে `Enter` চাপুন:

```bash
npm run dev
```

৫. কমান্ডটি রান হওয়ার পর টার্মিনালে একটি লোকাল লিংক দেখতে পাবেন, যেমন: `http://localhost:5173/` 
৬. আপনার কীবোর্ডের `Ctrl` বাটন চেপে ধরে ওই লিংকে ক্লিক করুন অথবা লিংকটি কপি করে আপনার ব্রাউজারে (Chrome, Edge ইত্যাদি) পেস্ট করে `Enter` চাপুন।

আপনার ওয়েবসাইটটি এখন আপনার কম্পিউটারে সফলভাবে চালু হয়েছে!

## ৩. প্রোডাকশন বিল্ড (Production Build) - যদি লাইভ সার্ভারে আপলোড করতে চান

আপনি যদি ওয়েবসাইটটি ভবিষ্যতে কোনো হোস্টিং সার্ভারে (যেমন cPanel, Vercel, Netlify) আপলোড করতে চান, তবে নিচের কমান্ডটি ব্যবহার করুন:

```bash
npm run build
```

এই কমান্ডটি রান করার পর `dist` নামে একটি নতুন ফোল্ডার তৈরি হবে। ওই ফোল্ডারের ভেতরের ফাইলগুলোই মূলত আপনার আসল ওয়েবসাইট যা সার্ভারে আপলোড করতে হবে।
